## Cosmic Launcher v0.1.0 (Alpha)

Welcome to the initial release of **Cosmic Launcher**! 🌌

Cosmic Launcher is a lightweight, custom-built desktop application designed specifically to organize and launch your favorite DRM-free games. No bloat, no background verification tracking—just click and play.

### Features
* **DRM-Free Management:** Easily launch standalone executables without external platform restrictions.
* **Lightweight Performance:** Built on Tauri, ensuring near-zero RAM usage while idle.
* **Fresh Aesthetics:** Updated with our brand-new custom cosmic rocket icons.

### Installation Instructions
1. Download **`Cosmic Launcher_0.1.0_x64-setup.exe`** or the `.msi` package below.
2. Run the installer and follow the desktop setup prompts.
3. Launch and enjoy!
